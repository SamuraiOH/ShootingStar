# ShootingStar -流れボッシー-
Unityの個人製作

## はじめに
　このリポジトリ上のプロジェクトには、利用規約の関係で背景画像、音声素材とそのメタファイルは含まれておりません。あらかじめご了承ください。また、このプロジェクトはAndroidStudioを介してビルドしました。
